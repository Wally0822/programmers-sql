# SELECT

## Level 1

### 🔻 과일로 만든 아이스크림 고르기

```sql
SELECT 
    FH.FLAVOR
FROM 
    FIRST_HALF AS FH
JOIN 
    ICECREAM_INFO AS II
    ON FH.FLAVOR = II.FLAVOR
WHERE 
    FH.TOTAL_ORDER > 3000
    AND II.INGREDIENT_TYPE = 'fruit_based'
ORDER BY 
    FH.TOTAL_ORDER DESC;
```


```sql
SELECT FLAVOR
FROM (
    SELECT * 
    FROM FIRST_HALF 
    WHERE TOTAL_ORDER > 3000
) AS FH
JOIN ICECREAM_INFO AS II 
    ON FH.FLAVOR = II.FLAVOR
WHERE II.INGREDIENT_TYPE = 'fruit_based'
ORDER BY FH.TOTAL_ORDER DESC;
```
